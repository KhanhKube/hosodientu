HƯỚNG DẪN IMPORT LANDING PAGE EHOLDINGS LÊN WORDPRESS
==========================================================

FILE TRONG THƯ MỤC
------------------
1) index.html
   - Mở trực tiếp bằng trình duyệt để xem trước (web + mobile).

2) eholdings-landing-wordpress.txt  ★ FILE CHÍNH ĐỂ DÙNG WP
   - Dán vào trang WordPress (Gutenberg / Custom HTML).

3) eholdings-elementor-html.txt
   - Dùng với Elementor: widget HTML → dán toàn bộ.

4) eholdings-wordpress-import.xml
   - Import qua Tools → Import → WordPress (tạo sẵn 1 trang).


CÁCH 1 — NHANH NHẤT (Khuyến nghị)
---------------------------------
1. Đăng nhập WordPress (quyền Administrator).
2. Vào: Trang → Thêm trang mới.
3. Tiêu đề trang: Hồ sơ điện tử của bạn
4. Bấm ⋮ (ba chấm) góc trên phải → chọn "Code editor"
   (hoặc thêm khối "Custom HTML").
5. Mở file: eholdings-landing-wordpress.txt
6. Copy từ dòng: <!-- wp:html -->
   đến hết file (bao gồm <!-- /wp:html -->).
7. Dán vào trang → Xuất bản.
8. Cài template trang = "Full width" / "Không sidebar"
   (tùy theme: Astra, Flatsome, Kadence, GeneratePress...).
9. Xem trang trên desktop và điện thoại.


CÁCH 2 — ELEMENTOR
------------------
1. Tạo trang mới → Edit with Elementor.
2. Kéo widget HTML vào canvas.
3. Mở file eholdings-elementor-html.txt → Copy toàn bộ → Paste.
4. Publish.
5. Page Layout: Elementor Full Width / Canvas (nếu muốn bỏ header theme).


CÁCH 3 — IMPORT XML
-------------------
1. Vào: Công cụ (Tools) → Import → WordPress.
2. Cài importer nếu chưa có → Run Importer.
3. Chọn file: eholdings-wordpress-import.xml → Upload.
4. Assign author → Submit.
5. Vào Trang → tìm "Hồ sơ điện tử của bạn" → Sửa / Xem.


NẾU COUNTDOWN / CAROUSEL / ACCORDION KHÔNG CHẠY
-----------------------------------------------
Một số theme hoặc security plugin lọc thẻ <script>.
Cách xử lý:
1. Cài plugin "Insert Headers and Footers" (WPCode).
2. Mở eholdings-landing-wordpress.txt, copy phần trong <script>...</script>.
3. Dán vào Footer scripts của plugin → Save.


GỢI Ý HIỂN THỊ ĐẸP HƠN
----------------------

- Ẩn title mặc định của theme nếu bị trùng tiêu đề.

- Hotline / tên học viên / mã hồ sơ: tìm & thay trong file .txt.
